Syllabus

This is still a tentative syllabus under development since this is a new decal starting in Fall 2020.

W Date Lecture Topic Homework Description
1 09/07 - 09/12 Introduction: Autonomous Driving and RC Car System Safety
  • Intro to modern theory of autonomous driving and safely operating RC cars
  • Show vehicle assembly and disassembly video
2 09/14 - 09/18 Arduino Circuit simulator
  • Introduction to Arduino, an open-source electronics platform
3 09/21 - 09/25 Jetson Nano and OpenCV AruCo marker detection
  • Introduction to Jetson Nano, a pocket-size platform for AI
  • Introduction to OpenCV, an open-source library of computer vision algorithms
4 09/28 - 10/02 Computer Vision: Localization AruCo marker location
  • Intro to computer vision and rigid-body motion
  • Calculate real-time localization
5 10/05 - 10/09 Computer Vision: Obstacle Detection None
  • Use depth cameras to detect and segment on-road obstacle images
6 10/12 - 10/16 Virtual Reality Racing Roll a ball
  • Intro to virtual reality programming
  • Implement a virtual cockpit app to remote control the RC car
7 10/19 - 10/23 3D Printing and Industry Design Make your own 3D name tag

  • Have hands-on experience with designing a 3D model.

  • Talk about ROAR Simulation Architecture

  • Write a PID control algorithm for waypoint navigation based on real-time localization
